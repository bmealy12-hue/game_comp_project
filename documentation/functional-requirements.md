# Functional Requirements

FR1 - The system shall display popular games from the RAWG API.

FR2 - The system shall allow users to search for games.

FR3 - The system shall allow users to filter games by genre.

FR4 - The system shall allow users to filter games by platform.

FR5 - The system shall display detailed information about a selected game.

FR6 - The system shall retrieve pricing information from IsThereAnyDeal.

FR7 - The system shall compare prices from multiple retailers.

FR8 - The system shall identify and display the cheapest available deal.

FR9 – The system shall provide users with a direct link to the retailer offering the selected deal.

FR10 - The system shall handle API errors gracefully.
